# Overview

A collection of LeetCode solutions and data structs implemented in C/C++.

# Prerequirement

## Format

Format the source code using clang-format (install via `brew install clang-format`), format CMake files using cmake-format (install via `pip install cmake-format`).

## Lint

Using cpplint for code linting (install via `pip install cpplint`).

# Dependencis

- Uthash 2.3.0
- GoogleTest 1.17.0
